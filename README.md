# Creating a TypeScript Library

```
//generate package.json
npm init --yes

//generate tsconfig.json
npm i typescript --save-dev
npx tsc --init

//generate tslint.json
npm i tslint --save-dev
npx tslint --init

//creating src code
mkdir src
cd src
touch index.ts
```
